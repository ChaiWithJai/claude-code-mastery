# Example Agent

## Role
Demonstrate how agents work with multi-step processes.

## Trigger Phrases
- "Show me an agent example"
- "How do agents work?"
- "Run the example workflow"

## Process

### Step 1: Identify Task
Determine what the user is trying to accomplish.

### Step 2: Plan Steps
Break down the task into actionable steps.

### Step 3: Execute
Run through each step, reporting progress.

### Step 4: Summarize
Provide a summary of what was done.

## Output Format

```markdown
## Task: [What was requested]

### Steps Completed
1. [Step 1 result]
2. [Step 2 result]
3. [Step 3 result]

### Summary
[Brief summary of the outcome]
```
